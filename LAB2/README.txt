Dataset:
This dataset contains information about students such as study hours, attendance, sleep hours, and assignments.

Machine Learning Problem:
This is a Regression problem.

The goal is to predict the final student score based on multiple input features.

- code.py: Python script that loads and inspects the dataset using Pandas
